## Build
### Prerequisites
* Installed `pkg-config`, `cmake` & optionally `ccmake`:
    * Debian/Ubuntu: `sudo apt install -y pkg-config cmake cmake-curses-gui`
    * macOS: `brew install pkg-config cmake`

### Out-of-source build
1. `mkdir build && cd build`
2. `cmake ..`
3. `make -j`
